A simple CLI stopwatch.


## Use

```
$ 4d -h
A simple CLI stopwatch.
Usage: 4d [DURATION]

4d		display elapsed time
4d 15m		countdown 15 minutes
4d 3h2m1s	countdown 3 hours, 2 minutes, 1 second

Ctrl-C exits.
```
